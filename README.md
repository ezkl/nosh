nosh
====

Description
-----------

A very simple Typhoeus/Nokogiri wrapper. Conceptually based on the eat library.

Usage
-----

    require 'nosh'

Requirements
------------
* Typhoeus (v0.2.4)
* Nokogiri (v1.4.6)

Install
-------

    $ gem install nosh
